# eLeaveForgotPassword-V1

Installation
------------
> $ npm install


Running the app
--------------

#### In developement mode
> $ ionic serve


Documentation
-------------

#### Build documentation
> $ npm run doc:build

#### Build and server documentation on http://localhost:8080
> $ npm run doc:serve
